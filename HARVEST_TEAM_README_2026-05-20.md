## What ships first

**Stall auto-terminator** (doc #1). It is the prerequisite for any meaningful prompt-engineering A/B — running prompt experiments against the current ~50% wasted-call baseline produces uninterpretable results. Ship the terminator, re-baseline the 12-session corpus on the same seeds, *then* run the doom-loop doc's Phase 1.

The terminator's refined rule (also in doc #1): terminate only if *(plateau length ≥ 25 turns) AND (shuffle fraction during the plateau ≥ 0.6)*. The shuffle-fraction gate prevents false positives like session `1279a3`, which has a 24-turn plateau but is honest early-game card-hunting via draws — it must not terminate.
